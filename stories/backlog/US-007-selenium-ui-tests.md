# US-007 - Selenium UI Functional Tests
## User Story
As a developer I want Selenium tests for key UI workflows so that I can verify end-to-end functionality.
## Acceptance Criteria

- [ ] Test verifies task creation via UI
- [ ] Test verifies task editing via UI
- [ ] Test verifies task deletion via UI
- [ ] Tests run against live Docker stack
- [ ] Tests triggered from command line
