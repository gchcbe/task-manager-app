# US-002 - Create a Task
## User Story
As a user I want to create a new task so that I can track things I need to do.
## Acceptance Criteria
- [x] New Task button visible in header
- [x] Clicking opens a form modal
- [x] Form has title (required), description, status and due date fields
- [x] Submitting creates task and closes modal
- [x] New task appears immediately in list
- [x] Validation error shown if title is empty
## Status: Done
