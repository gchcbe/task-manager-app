# US-003 - Edit a Task
## User Story
As a user I want to edit an existing task so that I can update its details or status.
## Acceptance Criteria
- [x] Each task card has an Edit button
- [x] Edit opens form pre-filled with current data
- [x] User can modify all fields
- [x] Submitting saves changes and closes modal
- [x] Updated task reflected immediately in list
## Status: Done
