# US-005 - Filter Tasks by Status
## User Story
As a user I want to filter tasks by status so that I can focus on relevant tasks.
## Acceptance Criteria
- [x] Filter buttons shown for All, To Do, In Progress and Done
- [x] Each button shows count of tasks in that status
- [x] Clicking a filter shows only matching tasks
- [x] Active filter is visually highlighted
- [x] Counts update immediately after task changes
## Status: Done
