# US-001 - View All Tasks
## User Story
As a user I want to see all my tasks so that I can get an overview of everything I need to do.
## Acceptance Criteria
- [x] All tasks displayed on main screen on load
- [x] Each task shows title, description, status, due date and created date
- [x] Tasks displayed as cards in responsive grid
- [x] Empty state shown when no tasks exist
- [x] Loading indicator shown while fetching
## Status: Done
