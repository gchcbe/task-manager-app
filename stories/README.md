# User Stories — Task Manager App

## Definition of Done (applies to every story)
A story is only marked Done when ALL of the following are true:
1. Code written and working
2. Manually verified in the UI
3. Unit tests written and passing
4. Selenium tests written and passing
5. Committed and pushed to GitHub

## Workflow
- backlog/      - not yet started
- in-progress/  - currently being worked on
- done/         - ALL definition of done criteria met
